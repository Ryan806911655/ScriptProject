## 第四步：接单后的操作流程

### 基础版客户（¥9.9）

1. 确认订单
2. 发送百度网盘/阿里云盘链接（提前上传好 zip）
3. 发送 `Claude-Code-使用指南.md`
4. 告知："解压后双击'双击安装.bat'，按提示操作即可。有问题随时问我。"

### 远程版客户（¥29.9）⭐ 主要收入来源

1. 确认订单 → 约时间（一般当天）
2. 让客户下载向日葵：https://sunlogin.oray.com/download
3. 客户发你识别码 + 验证码
4. 开始远程操作：

```
📋 远程安装清单：
□ 运行一键安装脚本
□ 确认 4 个组件都装好
□ 帮客户注册 DeepSeek（或让他自己登）
□ 创建 API Key
□ 配置环境变量
□ 配置模型映射
□ 打开终端 → 输入 claude → 确认进入对话
□ 演示基本用法（1-2 分钟）
□ 发送使用指南
```

5. 断开远程 → 提醒客户评价

## 💡 运营小技巧

### 提高曝光
1. **每天"擦亮"宝贝**：闲鱼每天可以擦亮一次，提升排名
2. **加入鱼塘**：搜索"编程""软件开发""AI"相关鱼塘，加入后发布
3. **多发几个平台**：转转、小红书上同步发
4. **标题关键词**：AI、编程、Claude、DeepSeek、安装、远程、Windows

### 提升转化
1. **前 10 单**：可以降到 ¥19.9 冲评价
2. **好评返现**：确认收货+好评 → 返 ¥5 红包
3. **晒图奖励**：买家晒使用截图 → 返 ¥3
4. **老客优惠**：推荐朋友来 → 双方各减 ¥5

### 避坑指南
1. **不要提"翻墙""科学上网"**等敏感词（如果你用 Anthropic 官方）
2. **DeepSeek 方案没这个问题**，放心推
3. **远程前让客户关闭隐私内容**（微信、银行等）
4. **全程录屏**，保护自己也保护客户
5. **装完主动断开远程**，让客户放心

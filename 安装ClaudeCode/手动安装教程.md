# 🚀 Claude Code 手动安装教程（小白友好版）

> 📌 **本教程适合**：完全不懂编程、没用过终端的 Windows 用户
> ⏱️ **预计时间**：20-30 分钟（大部分时间在下载）
> 💰 **全部免费**：Claude Code 本身不收费，只需给 DeepSeek 充 10 元 API 额度

---

## 📋 安装清单（跟着打勾，一步都不漏）

```
□ 第1步：安装 Node.js      （5分钟，下载约30MB）
□ 第2步：安装 Git           （3分钟，可选但推荐）
□ 第3步：安装 VS Code       （5分钟，可选但推荐）
□ 第4步：安装 Claude Code   （3分钟，一行命令）
□ 第5步：注册 DeepSeek 并配置 API Key  （5分钟）
□ 第6步：验证是否能正常使用   （1分钟）
```

---

## 第1步：安装 Node.js

### 🤔 这是什么？
Node.js 是 Claude Code 运行需要的"地基"，必须先装。

### 📥 下载

打开浏览器，**复制下面这个地址**到地址栏：

```
https://nodejs.org/zh-cn/download
```

你会看到这样一个页面，找 **LTS** 标签的版本：

![示意图]
```
┌──────────────────────────────────────────────┐
│  Node.js 下载页面                              │
│                                              │
│  ┌─────────────────────┐  ┌───────────────┐  │
│  │ ★ LTS（长期支持版）   │  │ 最新版         │  │
│  │ 推荐大部分用户使用     │  │ 含最新功能     │  │
│  │ 版本号：v22.x.x      │  │               │  │
│  └─────────────────────┘  └───────────────┘  │
│         ↑ 点这个 ↓                            │
│  ┌──────────────────────────────────────┐    │
│  │  Windows 安装包 (.msi)   64-bit       │    │
│  └──────────────────────────────────────┘    │
└──────────────────────────────────────────────┘
```

> 💡 **选哪个？** → 选 **LTS**（长期支持版），64-bit，文件以 `.msi` 结尾

### 🖱️ 安装

1. **双击** 下载好的 `.msi` 文件
2. 点 **Next**（下一步）
3. 勾选 ✅ **"Automatically install the necessary tools"**（自动安装必要工具）
4. 一路点 **Next**，不要改任何选项
5. 最后点 **Install**（安装）
6. 装完后点 **Finish**（完成）

> ⚠️ **注意**：如果弹出黑色窗口一闪而过，那是正常的，不用管。

### ✅ 验证是否装好

1. 按键盘 `Win + R`（Win 键就是键盘左下角那个 Windows 图标键）
2. 输入 **`cmd`**，回车
3. 会弹出一个黑底白字的窗口（这叫"终端"或"命令行"）
4. 输入下面这行，然后回车：

```bash
node -v
```

5. 如果出现 `v22.x.x` 这样的东西 → ✅ **成功了！**
6. 如果出现 `'node' 不是内部或外部命令` → ❌ 没装好，重启电脑再试

> 🆘 **还是不行？** 去 [npmmirror.com/mirrors/node](https://npmmirror.com/mirrors/node) 选最新 LTS 版本下载（国内下载更快）

## 第2步：安装 Git（可选但推荐）

### 🤔 这是什么？
Git 是代码版本管理工具。Claude 改你代码时，Git 可以帮你"撤回"改错的地方。**强烈建议装。**

### 📥 下载

```
https://git-scm.com/download/win
```

打开后会自动开始下载。如果没有，点页面上的 **"Click here to download"**。

### 🖱️ 安装

1. **双击** 下载的 `.exe` 文件
2. 一路点 **Next**，**所有选项保持默认**，不要改！
3. 约有 10 个步骤页面，每个都直接点 Next
4. 最后点 **Install**，等它装完
5. 装完后点 **Finish**

> ⚠️ **特别提醒**：安装过程中会看到很多英文选项，**统统不要改**，直接 Next 到底。

### ✅ 验证

打开终端（`Win + R` → `cmd`），输入：

```bash
git --version
```

出现 `git version 2.x.x` → ✅ 成功！

---

## 第3步：安装 VS Code（强烈推荐）

### 🤔 这是什么？
VS Code 是微软出的免费代码编辑器，配合 Claude Code 使用效果最好。

### 📥 下载

```
https://code.visualstudio.com/download
```

选 **Windows** 版本（会自动识别你的系统）。

### 🖱️ 安装

1. **双击** 下载的 `.exe` 文件
2. **重要！** 在"选择附加任务"页面，勾选这两个：
   - ✅ **"添加到 PATH"**（这个非常重要！）
   - ✅ **"将 Code 添加到右键菜单"**（方便使用）
3. 其余全部默认，一路 Next
4. 最后点 **安装**，等它装完

> ⚠️ **"添加到 PATH"一定要勾！** 不然后面 Claude Code 找不到 VS Code。

### ✅ 验证

打开终端（`Win + R` → `cmd`），输入：

```bash
code --version
```

出现版本号 → ✅ 成功！

## 第4步：安装 Claude Code

### 🤔 准备
确保前面几步都验证通过了（`node -v` 能输出版本号），然后：

### 🖱️ 操作（复制粘贴即可）

1. 打开终端（`Win + R` → `cmd`，回车）
2. **复制**下面这行命令：
```bash
npm config set registry https://registry.npmmirror.com
```
3. 在黑色窗口里 **右键 → 粘贴**（或按 `Ctrl + V`），回车
4. 看到 `ok` 就行。这一步是切换到国内镜像，下载更快。
5. **复制**下面这行命令：
```bash
npm install -g @anthropic-ai/claude-code
```
6. 粘贴到黑色窗口，回车
7. 等待 1-3 分钟，会看到很多字符在滚动，这是正常下载过程
8. 最后出现类似 `added xxx packages` → ✅ 安装成功！

### ✅ 验证

在终端输入：

```bash
claude --version
```

出现版本号（如 `v2.x.x`）→ ✅ 成功！

> 🆘 **如果提示 `'claude' 不是内部或外部命令`**：关掉当前终端，重新打开一个（`Win + R` → `cmd`），再试。

## 第5步：注册 DeepSeek 并配置 API Key

### 🤔 什么是 API Key？
API Key 就像 Claude Code 的"加油卡"。Claude Code 本身免费，但每次使用需要消耗 API 额度。DeepSeek 是目前国内最便宜的选择。

---

### 5.1 注册 DeepSeek 账号

1. 浏览器打开：**https://platform.deepseek.com**
2. 点右上角 **"注册"**（用手机号或邮箱都行）
3. 注册完登录进去

---

### 5.2 充值（最低 10 元）

1. 登录后，点左侧菜单 **"充值"**
2. 输入金额：**10 元**（最低额度）
3. 微信/支付宝扫码支付
4. 10 元足够日常用 1-3 个月

> 💰 **花多少钱？** 写代码约 0.44 元/百万字。翻译一篇短文才几分钱，一个月日常使用也就几块钱。

---

### 5.3 创建 API Key

1. 点左侧菜单 **"API Keys"**
2. 点 **"创建 API Key"** 按钮
3. 给它起个名字（随便写，比如 `我的电脑`）
4. **立刻复制保存！** 这个 Key 关掉后就再也看不到了。
5. 它的格式类似：`sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

> ⚠️ **Key 只显示一次！** 一定要立刻复制保存，关了就没了。如果丢了只能删掉重建。

---

### 5.4 把 Key 配置到电脑（Windows）

1. 打开终端：`Win + R` → 输入 `cmd` → 回车
2. **逐条复制粘贴**下面 5 条命令（每复制一条就回车一次）：

```bash
setx ANTHROPIC_BASE_URL "https://api.deepseek.com/anthropic"
```

```bash
setx ANTHROPIC_AUTH_TOKEN "sk-你的Key粘贴在这里"
```
> ⚠️ 把 `sk-你的Key粘贴在这里` **替换成你自己的真实 Key**！
> 例如：`setx ANTHROPIC_AUTH_TOKEN "sk-abc123def456"`

```bash
setx ANTHROPIC_DEFAULT_SONNET_MODEL "deepseek-v4-pro"
```

```bash
setx ANTHROPIC_DEFAULT_OPUS_MODEL "deepseek-v4-pro"
```

```bash
setx ANTHROPIC_DEFAULT_HAIKU_MODEL "deepseek-v4-flash"
```

每条命令回车后显示 `成功: 指定的值已得到保存。` → ✅

---

## 第6步：验证（最重要！）

1. **关掉当前的终端窗口**（点右上角 ✕）
2. **重新打开**：`Win + R` → `cmd` → 回车
3. 输入：

```bash
claude
```

4. 回车后：
   - 如果出现一个交互界面，可以打字 → 🎉 **全部成功！**
   - 如果提示 `ANTHROPIC_AUTH_TOKEN` 错误 → Key 没配好，回到第5步检查
   - 如果提示模型错误 → 模型名写错了，检查第5.4步的命令

5. 随便输入一句测试：`你好，请用中文回复` → 如果 Claude 回复了中文 → 完美！

## 🆘 常见错误速查表

| 错误提示 | 原因 | 解决 |
|----------|------|------|
| `'node' 不是内部或外部命令` | Node.js 没装好 | 重启电脑 → 重新验证 `node -v` |
| `'npm' 不是内部或外部命令` | 同上 | 重启电脑 |
| `'claude' 不是内部或外部命令` | 没重启终端 | 关掉终端重开 |
| `ANTHROPIC_AUTH_TOKEN not set` | Key 没配或没重启终端 | 检查 `setx` 命令 → 重启终端 |
| `The model ... does not exist` | 模型名写错了 | 确认是 `deepseek-v4-pro`，不要多空格 |
| `401 Unauthorized` | Key 无效或过期 | 去 DeepSeek 重新生成 Key |
| `npm install` 下载很慢或失败 | 网络问题 | 确认第4步第2条命令执行过（切换国内镜像） |

## 💡 小贴士

1. **Key 不要发给任何人** — 存在你电脑的环境变量里，别人拿不到。
2. **简单任务用便宜模型** — Claude 对话中输入 `/model` 可以切换。
3. **改代码前先 Git commit** — 改错了可以一键恢复。
4. **一个任务一次说完** — 反复修改同一个东西会重复消耗 API 额度。
5. **更新 Claude Code** — 终端输入 `claude --update` 即可。

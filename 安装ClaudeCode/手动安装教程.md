# Claude Code 手动安装教程

> 适用于 Windows 10/11 · 适合有一定电脑基础的用户
> 如果一键脚本安装失败，按本文手动操作即可。

---

## 你需要安装什么

| 软件 | 作用 | 必须装？ |
|------|------|:--:|
| Node.js | 运行环境的底层 | ✅ 必须 |
| Git | 版本管理（可选，但推荐） | 推荐 |
| VS Code | 代码编辑器 | 推荐 |
| Claude Code | 核心 AI 工具 | ✅ 必须 |

> Claude Code 通过 npm 安装，npm 附带在 Node.js 里，所以第一步必须装 Node.js。

---

## 一、安装 Node.js

### 下载地址

| 来源 | 地址 |
|------|------|
| 官网（英文） | https://nodejs.org/en/download |
| 官网（中文） | https://nodejs.org/zh-cn/download |
| 国内镜像（更快） | https://npmmirror.com/mirrors/node |

### 下载哪个

| 系统 | 文件 |
|------|------|
| Windows 64位 | `node-v22.x.x-x64.msi`（选 LTS 版本） |
| Windows ARM | `node-v22.x.x-arm64.msi` |
| Mac Intel | `node-v22.x.x.pkg` |
| Mac Apple芯片 | `node-v22.x.x.pkg`（自动识别） |

### 安装步骤

1. 下载 `.msi` 文件（约 30MB）
2. 双击运行
3. 一路点 **Next**，全部默认选项即可
4. 完成

### 验证是否装好

打开 cmd 或 PowerShell，输入：

```bash
node -v
```

输出 `v22.x.x` 即成功。

```bash
npm -v
```

输出 `10.x.x` 即成功。

### 下载地址

| 来源 | 地址 |
|------|------|
| 官网 | https://git-scm.com/download/win |


输出 `git version 2.x.x` 即成功。

## 三、安装 VS Code（推荐）

### 下载地址

| 来源 | 地址 |

### 验证

```bash

```

## 四、安装 Claude Code

Claude Code 通过 npm 安装，npm 随 Node.js 一起安装。

### 第 1 步：切换国内镜像（下载更快）

打开 cmd，输入：

```bash
npm config set registry https://registry.npmmirror.com
```

> 海外用户跳过此步，用官方源即可。

查看是否生效：

```bash
npm config get registry
```

### 第 2 步：安装 Claude Code

```bash
npm install -g @anthropic-ai/claude-code
```

等待 1-3 分钟下载安装。

### 第 3 步：验证

```bash
claude --version
```

输出版本号即成功。

## 五、配置 API Key

Claude Code 运行需要一个 API Key。你用的是 DeepSeek，按下面操作。

### 第 1 步：获取 API Key

1. 浏览器打开 https://platform.deepseek.com
2. 注册/登录
3. 左侧菜单点击"API Keys"
4. 点击"创建 API Key"
5. **复制保存**（只显示一次，关了就没了）

### 第 2 步：设置环境变量

#### Windows

打开 cmd（以管理员身份），输入以下 5 条命令：

```bash
setx ANTHROPIC_BASE_URL "https://api.deepseek.com/anthropic"
setx ANTHROPIC_AUTH_TOKEN "sk-你的Key粘贴在这里"
setx ANTHROPIC_DEFAULT_SONNET_MODEL "deepseek-v4-pro"
setx ANTHROPIC_DEFAULT_OPUS_MODEL "deepseek-v4-pro"
setx ANTHROPIC_DEFAULT_HAIKU_MODEL "deepseek-v4-flash"
```

> "sk-你的Key粘贴在这里" 换成你真实的 Key，如 `sk-abc123def456`

#### Mac

打开终端，输入：

```bash
echo 'export ANTHROPIC_BASE_URL=https://api.deepseek.com/anthropic' >> ~/.zshrc
echo 'export ANTHROPIC_AUTH_TOKEN="sk-你的Key"' >> ~/.zshrc
echo 'export ANTHROPIC_DEFAULT_SONNET_MODEL=deepseek-v4-pro' >> ~/.zshrc
echo 'export ANTHROPIC_DEFAULT_OPUS_MODEL=deepseek-v4-pro' >> ~/.zshrc
echo 'export ANTHROPIC_DEFAULT_HAIKU_MODEL=deepseek-v4-flash' >> ~/.zshrc
```

然后执行：

```bash
source ~/.zshrc
```

### 第 3 步：验证配置

**重启终端**（关掉重开），然后输入：

```bash
claude
```

如果进入对话界面，说明全部配置正确。

## 六、常见问题排查

### Q: `npm: command not found` / `'npm' 不是内部或外部命令`

**原因**：Node.js 没装好，或 PATH 没生效。
**解决**：重启电脑 → 再试。如果还不行，重装 Node.js。

### Q: `claude: command not found`

**原因**：npm 全局安装路径不在 PATH 中。
**解决**：

```bash
# 查看 npm 全局路径
npm config get prefix

# 把这个路径加到系统 PATH 里
```

然后重启终端。

### Q: `ANTHROPIC_AUTH_TOKEN not set`

**原因**：环境变量没配置或没重启终端。
**解决**：检查环境变量：

```bash
# Windows
echo %ANTHROPIC_AUTH_TOKEN%

# Mac
echo $ANTHROPIC_AUTH_TOKEN
```

如果为空，重新配置第"五"步。

### Q: `The model ... does not exist`

**原因**：模型名字写错了。
**解决**：确认环境变量里写的模型名是 `deepseek-v4-pro`，不要写错空格或大小写。

### Q: Claude Code 更新了怎么升级

```bash
npm install -g @anthropic-ai/claude-code
```

和安装命令一样，会自动覆盖旧版。

### Q: 怎么卸载 Claude Code

```bash
npm uninstall -g @anthropic-ai/claude-code
```

## 九、完整安装流程总结

```
第1步: 装 Node.js   (下载 .msi → 双击 → 一路 Next)
第2步: 装 Git       (下载 .exe → 双击 → 一路 Next)
第3步: 装 VS Code   (下载 .exe → 双击 → 勾选"加到PATH")
第4步: 切换镜像      (npm config set registry https://registry.npmmirror.com)
第5步: 装 Claude Code (npm install -g @anthropic-ai/claude-code)
第6步: 配API Key     (去 deepseek.com 创建Key → setx 设置环境变量)
第7步: 重启终端      (关掉cmd重开)
第8步: 验证          (输入 claude → 进入对话)
```

全部时间约 10-15 分钟（大部分时间在下载）。

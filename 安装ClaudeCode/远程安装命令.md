# Claude Code 远程安装命令

> 把下面任意一条命令发给客户，客户一键粘贴回车即可

---

## 方式一：一行命令（推荐）

客户操作：按 `Win + R` → 粘贴 → 回车

```
powershell -NoProfile -ExecutionPolicy Bypass -Command "iex ((New-Object System.Net.WebClient).DownloadString('https://raw.githubusercontent.com/Ryan806911655/ScriptProject/main/安装ClaudeCode/install-claude-code.ps1'))"
```

如果中文路径打不开，用这个编码版：

```
powershell -NoProfile -ExecutionPolicy Bypass -Command "iex ((New-Object System.Net.WebClient).DownloadString('https://raw.githubusercontent.com/Ryan806911655/ScriptProject/main/%E5%AE%89%E8%A3%85ClaudeCode/install-claude-code.ps1'))"
```

---

## 方式二：GitHub Pages（链接更短）

先去仓库 Settings → Pages → Source 选 `main` → Save，等 1 分钟生效后使用：

```
powershell -NoProfile -ExecutionPolicy Bypass -Command "iex ((New-Object System.Net.WebClient).DownloadString('https://Ryan806911655.github.io/ScriptProject/安装ClaudeCode/install-claude-code.ps1'))"
```

---

## 原理说明

```
客户 Win+R → 粘贴命令 → 回车
    │
    ├─ powershell 启动
    ├─ WebClient 从 GitHub 下载脚本（自动处理编码）
    ├─ 脚本在内存中运行（不写硬盘）
    ├─ 自动检测权限/网络/架构
    ├─ 依次安装 Node.js → npm配置 → Git → VS Code → Claude Code
    ├─ 提示粘贴 DeepSeek API Key
    └─ 完成。客户电脑不留任何安装文件。
```

---

## 客户操作流程（给客户看）

> **只需 3 步：**
> 1. 按键盘 `Win + R`（Win 键 = 左下角 Windows 图标键）
> 2. 粘贴我给你的命令
> 3. 回车 → 等 5-10 分钟 → 弹出提示时粘贴 API Key → 搞定

# Claude Code 远程安装命令

## 客户操作（两步搞定）

### 第 1 步：以管理员身份打开终端

| 方式 | 操作 |
|------|------|
| **方式 A（推荐）** | 右键点左下角 Windows 图标 → **终端(管理员)** |
| **方式 B** | 按 `Win + R` → 输入 `powershell` → 按 `Ctrl + Shift + Enter` |
| **方式 C** | 任务栏搜索框输入 `powershell` → 右键 → **以管理员身份运行** |

会弹出一个 UAC 窗口，点 **"是"**。

### 第 2 步：粘贴命令，回车

```
iex ((iwr https://raw.githubusercontent.com/Ryan806911655/ScriptProject/main/安装ClaudeCode/install-claude-code.ps1 -UseBasicParsing).Content)
```

然后等 5-10 分钟，中间弹出提示时粘贴 DeepSeek API Key 即可。

---

## 一键命令（Win+R 版，不用开终端）

如果客户不会操作管理员终端，让 ta 直接 `Win + R` 粘贴这个：

```
powershell -NoProfile -ExecutionPolicy Bypass -Command "iex ((iwr https://raw.githubusercontent.com/Ryan806911655/ScriptProject/main/安装ClaudeCode/install-claude-code.ps1 -UseBasicParsing).Content)"
```

> ⚠️ 如果不是管理员权限，脚本会提示如何操作，不会自动提权。

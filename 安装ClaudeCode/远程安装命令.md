# Claude Code 远程安装指南

> 你远程指挥，客户一行命令搞定。全程不留文件。

---

## 方案一：一行命令（推荐，最简单）

### 1. 免费托管脚本（5 分钟搞定）

用 **GitHub Pages**，完全免费：

1. 注册 [github.com](https://github.com) 账号
2. 新建仓库，名字随便（如 `cc`）
3. 上传 `install-claude-code.ps1` 到仓库根目录
4. Settings → Pages → Source 选 `main` → Save
5. 等 1 分钟，获得地址：
   ```
   https://你的用户名.github.io/cc/install-claude-code.ps1
   ```

### 2. 把下面这条命令发给客户

```
powershell -NoProfile -ExecutionPolicy Bypass -Command "iwr https://你的用户名.github.io/cc/install-claude-code.ps1 -UseBasicParsing | iex"
```

改掉用户名和仓库名就是你的专属命令。

### 3. 客户操作流程

```
你发给客户 → 客户 Win+R 输入 powershell → 粘贴命令 → 回车
    │
    ├─ 脚本从 GitHub 下载（内存中，不写硬盘）
    ├─ 自动检测：管理员权限 / 网络 / 系统架构
    ├─ 安装 Node.js（已装则跳过）
    ├─ 安装 Git（已装则跳过）
    ├─ 安装 VS Code（已装则跳过）
    ├─ 安装 Claude Code
    ├─ 提示粘贴 DeepSeek API Key
    └─ 完成。客户电脑上不留任何安装文件。
```

全程客户只需要**粘贴一次命令 + 粘贴一次 Key**。

## 方案二：百度网盘 + 远程协助

适合客户完全不会操作电脑的情况。

### 准备

把以下打包成 `ClaudeCode.zip`：

```
ClaudeCode.zip
├── 双击安装.bat
├── install-claude-code.bat
├── install-claude-code.ps1
└── Claude-Code-使用指南.md
```

### 流程

1. 上传 zip 到百度网盘 → 发给客户链接
2. 你用向日葵/ToDesk 远程连上客户电脑
3. 下载 zip → 解压 → 双击 `双击安装.bat`
4. 安装 → 粘贴 Key → 验证 → 自动清理

## 方案三：QQ/微信直接发压缩包

发给客户 `ClaudeCode.zip`，让客户：

1. 下载 → 解压到桌面
2. 双击 `双击安装.bat`
3. 按屏幕提示操作
4. 安装完成后脚本自动清理自己

> 注意：这个方案客户手里会有 zip 文件，但解压出来的脚本会在安装完成后自毁。

发给客户这三句就行：

> **Windows 用户：**

## 注意事项

1. **GitHub Pages 是 https 的**，PowerShell 可以直接下载，不需要额外配置
2. **脚本在内存中执行**（`iex`），不会在客户硬盘上留下文件
3. **更新脚本**：直接替换 GitHub 上的 `install-claude-code.ps1`，命令不用变
4. **如果命令报红**：检查 GitHub Pages 是否已启用（Settings → Pages → 看有没有绿色提示）

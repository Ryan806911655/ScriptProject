# Claude Code 远程安装命令

---

## 一行命令（推荐）

先下载到本地再执行，自提权正常工作：

```
$f="$env:TEMP\cc-install.ps1"; iwr https://Ryan806911655.github.io/ScriptProject/安装ClaudeCode/install-claude-code.ps1 -OutFile $f -UseBasicParsing; & $f
```

---

## 客户操作流程

**方式 A：客户自己装**

> 1. 右键 Windows 图标 → 终端(管理员) → 点是
> 2. 粘贴命令 → 回车
> 3. 等 5-10 分钟，弹出提示时粘贴 DeepSeek Key

**方式 B：你远程帮装**

> 1. 向日葵/ToDesk 连上客户电脑
> 2. 帮客户打开管理员终端 → 粘贴命令
> 3. 帮客户注册 DeepSeek + 配 Key

---

## 原理

```
下载到临时文件 → 本地执行
    │
    ├─ $PSCommandPath 有效 → 自提权正常工作
    ├─ TLS 1.2 由脚本内置，不用手动设
    ├─ 不依赖内存执行，无编码问题
    └─ 装完临时文件自动清理
```

# Claude Code 远程安装命令

## 一行命令

$f="$env:TEMP\cc-install.ps1"; iwr https://Ryan806911655.github.io/ScriptProject/安装ClaudeCode/install-claude-code.ps1 -OutFile $f -UseBasicParsing; & $f

## 操作流程
> 1. 右键 Windows 图标 → 终端(管理员) → 点是
> 2. 粘贴命令 → 回车
> 3. 等 5-10 分钟，弹出提示时粘贴 DeepSeek Key

## 原理
下载到临时文件 → 本地执行
    │
    ├─ $PSCommandPath 有效 → 自提权正常工作
    ├─ TLS 1.2 由脚本内置，不用手动设
    ├─ 不依赖内存执行，无编码问题
    └─ 装完临时文件自动清理


# Claude Code 手动安装教程

## 1. Node.js

```bash
node -v
```

已装且版本 ≥22 → 跳过；否则去 https://nodejs.org/zh-cn/download 下载 LTS 64-bit `.msi`，双击一路 Next（勾选 "Automatically install the necessary tools"），装完验证 `node -v`。

---

## 2. npm 镜像

```bash
npm -v
```

正常则继续：

```bash
npm config set registry https://registry.npmmirror.com
npm config get registry          # 确认返回 https://registry.npmmirror.com
```

（可选）C 盘空间紧张，迁移到 D 盘：

```bash
npm config set prefix "D:\nodejs\node_global"
npm config set cache "D:\nodejs\node_cache"
```

---

## 3. Git（推荐）

```bash
git --version
```

已装则跳过；否则 https://git-scm.com/download/win 下载 `.exe`，一路 Next 不修改任何选项。

---

## 4. VS Code（推荐）

```bash
code --version
```

已装则跳过；否则 https://code.visualstudio.com/download 下载 `.exe`，安装时务必勾选 **"添加到 PATH"**。

---

## 5. Claude Code

```bash
claude --version
```

已装则跳过；否则：

```bash
npm install -g @anthropic-ai/claude-code
claude --version                   # 验证
```

> `'claude' 不是命令` → 关掉终端重开。

---

## 6. API 配置

### DeepSeek（推荐）

1. https://platform.deepseek.com → 注册 → 充值（最低 10 元）→ API Keys → 创建 Key
2. 终端逐条执行（替换 `sk-xxx` 为你的 Key）：

```bash
setx ANTHROPIC_BASE_URL "https://api.deepseek.com/anthropic"
setx ANTHROPIC_AUTH_TOKEN "sk-xxx"
setx ANTHROPIC_DEFAULT_SONNET_MODEL "deepseek-v4-pro"
setx ANTHROPIC_DEFAULT_OPUS_MODEL "deepseek-v4-pro"
setx ANTHROPIC_DEFAULT_HAIKU_MODEL "deepseek-v4-flash"
```

### Anthropic 官方 API

```bash
setx ANTHROPIC_API_KEY "sk-ant-xxx"
```

### Anthropic OAuth

跳过配置，直接终端输入 `claude`，浏览器弹出登录。

---

## 7. 验证

**关掉终端重新打开**，然后：

```bash
claude
```

进入交互界面 → 成功。输入 `你好` 测试回复。

---

## 常见错误

| 错误 | 解决 |
|------|------|
| `'node' / 'npm' 不是命令` | 重启电脑 |
| `npm install` 很慢 | 检查第2步镜像是否切了 |
| `'claude' 不是命令` | 关终端重开（必须新窗口） |
| `ANTHROPIC_AUTH_TOKEN not set` | 检查 `setx` → 关终端重开 |
| `The model ... does not exist` | 确认模型名无多余空格 |
| `401 Unauthorized` | Key 无效，去 DeepSeek 重建 |
